# Whiteboard App

Online collaborative whiteboard application built with Next.js and tldraw.

## Features

- Create new whiteboards with unique IDs
- Join existing boards by ID
- Drawing, shapes, text, and arrows
- Share boards via URL
- Clean, responsive UI with Tailwind CSS

## Getting Started

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm start
```

## Tech Stack

- **Next.js 14** - React framework
- **tldraw** - Whiteboard canvas library
- **Tailwind CSS** - Styling
- **TypeScript** - Type safety
